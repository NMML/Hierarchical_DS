Welcome to hierarchicalDS.

Welcome to the developer page for hierarchicalDS, an R package designed to facilitate hierarchical analysis of distance sampling data from line transect studies (the code has not yet been expanded to allow estimation of point count data).

Here, you will find the latest stable (well, at least as far as I can tell) source code (check out NEWS for release notes). Unfortunately, Github no longer hosts certain types of files; for a vignette and binaries, please visit
